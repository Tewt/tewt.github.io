Listen to me.

A faint echo in the wind, barely audible, seems to be asking you a question. You try with all your might to listen to the voice, questioning whether or not it exists. The next time the question is asked you are able to hear it: Do you want to create your own website? You are able to decipher (roughly) the instructions that follow:

First, navigate to your profile on github.com. Observe the 'plus' symbol in the top right. Click on it, and then click on 'new repository'. Enter in a repository name and then click on 'create repository'. Next you will need to clone the repository on your machine. Copy the url under 'HTTPS clone URL'. Open up the terminal and go to your projects folder by typing 'cd projects'. Now type git clone and then paste the url you copied from git. Hit 'enter'. You should now see the name of the new repository you just created when you type 'ls'. Type 'cd repositoryname'. You should now see a file called 'index.html'. To apply stylings to this file, create a file called 'mystyle.css' (for example). To create new folders for the different pages on your site, type 'mkdir titleofthatpage'. Keep in mind you must create new html files for each new page. Enjoy your new website.

The wind begins to blow violently and then settle suddenly. You can no longer hear the voice.
